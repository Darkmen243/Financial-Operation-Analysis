# Отчет по анализу финансовых транзакций

## Введение
Был проведен анализ данных о финансовых транзакциях (10 000 записей) и клиентах (9 799 записей) за период с 01.2025 по 03.2025. 
Цель работы - очистка данных, анализ ключевых метрик, объединение данных и прогнозирование выручки.

## Выполненные задачи:

# 1. Очистка и подготовка данных

#### Транзакции:
- Удалены строки с пропущенными `transaction_id`, `amount`, `transaction_date` (474 записи)
- Заполнены пропуски в полях:
  - `client_id` → "Unknown" (202 записи)
  - `payment_method` → "Unknown" (заполнено)
  - `city` → "Unknown" (заполнено)
  - `service` → "other" (заполнено)
- Удалены транзакции с отрицательной суммой (460 записей)
- Приведены даты к формату datetime, удалены некорректные даты

#### Клиенты:
- Удалены строки с пропущенными `client_id` или `net_worth` (475 записей)
- Очищен возраст: удалены значения вне диапазона 0-120 лет (983 записи)
- Приведен пол к единому формату (male/female/unknown)
- Удалены дубликаты клиентов по `client_id`

### Результат очистки:
- **Транзакции:** с 10 000 → 9 064 записей
- **Клиенты:** с 9 799 → 8 273 записей

# 4. **Визуализация**
 ### Построенные графики:

1. **Распределение сумм транзакций** (гистограмма до 95-го перцентиля)
   - Выявлена правосторонняя асимметрия (много мелких транзакций)
   - 95% транзакций имеют сумму до ~150,000

2. **Выручка по типам услуг** (горизонтальная столбчатая диаграмма)
   - Инвестиционное консультирование - лидер с большим отрывом
   - Топ-5 услуг дают ~70% всей выручки

3. **Зависимость среднего чека от возраста** (линейный график со сглаживанием)
   - Наблюдается пик в возрасте 40-45 лет

4. **Способы оплаты** (круговая диаграмма)
   - Кредитная карта - основной способ (40%)
   - Криптовалюта занимает 5.3% - тренд на цифровые платежи


# 5. **Прогнозирование**

### Методология:
- **Модель:** Random Forest Regressor (ансамбль из 100 деревьев решений)
- **Признаки для прогноза:**
  - День недели (0-6)
  - Месяц (1-12)
  - Выходной день (бинарный признак)
  - Выручка за предыдущий день (Lag 1)
  - Выручка за предыдущую неделю (Lag 7)
  - Скользящая средняя за 7 дней (MA7)
   - ### Прогноз на следующий месяц (2025-03-21 - 2025-04-19):
   
     - Суммарная выручка: 144,791,962.65
     - Средний дневной доход: 4,826,398.76
     - Минимальный прогноз: 4,503,146.80
     - Максимальный прогноз: 5,609,922.99

    - ### Сравнение с прошлым месяцем:
  - Было: 147,007,796.96
  - Будет: 144,791,962.65

  -**Изменение: -1.5%**

# 6. Общие выводы и рекомендации

### Ключевые выводы:
1. **Самая популярная услуга:** Инвестиционное консультирование (21.9% заказов)
2. **Самая доходная услуга:** Инвестиционное консультирование (52.3% выручки)
3. **Ключевые клиенты:** с высоким капиталом (>1M) приносят 52.3% выручки
4. **Лучший город по среднему чеку:** Michellehaven (58,156.83)
5. **Основной способ оплаты:** Кредитная карта (40% транзакций)

## Рекомендации для бизнеса:
1. **Развивать услугу "Инвестиционное консультирование"** - она ключевая для выручки
2. **Усилить работу с клиентами с высоким капитолом** - они самые доходные
3. **Расширять присутствие в городах-лидерах** (Michellehaven, East Rachelmouth)
4. **Развивать безналичные способы оплаты** - доля наличных всего 9.9%
5. **Изучить рост популярности криптовалюты** (5.3% и растет)
